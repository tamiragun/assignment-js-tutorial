Basic syntax

Variables

In ES6, there are two ways to assign a variable: const or let. 'const' means that the identifier can't be reassigned. With 'let', you can reassign the variable afterwards. This makes 'const' a good candidate for variables that are unlikely to need to be reassigned (this does not mean that they are immatable though), and 'let' is a good candidate for declaring variables in loops or mathematical algorithms.

const name = "Tamira";
let car = "Hyundai";
console.log("Hello World!");

Conditional statements

Your basic if, else if and else statements look very similar to other languages such as Java and PHP. Note that "else if" is written as two separate words. Make sure the conditional is in brackets, and the statement in curly brackets. End every statement with a semicolon. The AND operator is a double "&&" and the OR operator is a double "||". You don't need a semicolon after the curly bracket close.

if (name === "Tamira" && car === "Hyundai") {
  console.log("Tamira has a Hyundai.");
}
else if (name === "Tamira" || car === "Hyundai") {
  console.log("Someone has a Hyundai.");
}
else {
  console.log("No one has a Hyundai");
}
console.log("Hello World!");

Arrays and loops

In JavaScript, arrays can contain elements of different types, and their length can change. Arrays are declared with square brackets surrounding the elements separated by commas. There are several ways to loop over an array in JavaScript. You can use the basic for loop, while loop, or array-specific methods such as for (x in y) and .forEach(). Below is an eaxample of each method, printing the different ages of a given family's family members:

let familyAges = [12,14,43,44];

for (let i=0; i < familyAges.length; i++) {
  console.log(familyAges[i]);
}

let i = 0;
while (i < familyAges.length) {
  console.log(familyAges[i]);
i++;
}

for (let age of familyAges) {
  console.log(age);
}

familyAges.forEach(function(age) {
  console.log(age);
});
console.log("Hello World!");

Maps

Maps are key value pairs in JavaScript. To create one, instatiate a Map object and the set the keys and values on the instance of that Map object. The Map class has a number of built-in functions that you can use to manipulate its contents.

let familyMembers = new Map();

familyMembers.set("Mother", "Sally");
familyMembers.set("Father", "Nkosi");
familyMembers.set("Son", "Jason");
familyMembers.set("Daughter", "Ungwang");

//Print the keys and values:
for (let [key, value] of familyMembers) {
  console.log(key + ' name is ' + value);
}

for (let [key, value] of familyMembers.entries()) {
  console.log(`${key} name is ${value}`);
}

//Print the keys only:
for (let key of familyMembers.keys()) {
  console.log(key);
}

//Print the values only:
for (let value of familyMembers.values()) {
  console.log(value);
}
console.log("Hello World!");

Built-in functions

JavaScript has a ton of built-in functions that allows you to get or manipulate information about your objects, strings, integers, floats, arrays, etc. Needless to say, Goole is your best friend here, but for now, check out this curated shortlist of handy built-in functions:

Function What it does
min() Returns the smallest valued element
max() Returns the largest value element
round() Returns a number rounded to the nearest integer
toUpperCase() Returns the given string all in upper case
toLowerCase() Returns the given string in all lower case
charAt() Returns a character in a string at the given index

User-defined functions

These built-in functions cover a lot of ground, but you will likely need more custom functions for the programmes you are building. JavaScript, like most other programming languages, allows you to define your own functions. With ES6, there are now more ways to define functions.

//Traditional function:
function doubleNumber (number) {
  return number * 2;
}

//Arrow function:
(number) => {
  return number * 2;
}

//Or even:
number => number * 2;

//With multiple arguments:
(number1, number2) => number1 * number2;

//With no arguments:
() => 2 * 2;

//To name your function:
const doubleNumber = (number) => number * 2;
console.log("Hello World!");
