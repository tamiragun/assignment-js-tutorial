Events

The real power of JavaScript lies in its ability to render web pages dynamic. One of the ways it can do this is by 'listening' to certain events that might happen on the page, and then executing code specific to that event.

Generally, you would select an element from the DOM, add an event listener, and then attach a function that must execute when the event is triggered.

//Shows a text in another DOM element upon button click:
let button = document.querySelector(".btn");
button.addEventListener("click", () => {
  document.getElementById("demo").innerHTML = "Hello World";
};

//You can also use the shorthand for some events:
button.onclick = function(){
  document.getElementById("demo").innerHTML = "Hello World";
};

//And you can even embed the script in the html:
<element onclick="myScript">
console.log("Hello World!");

Here's a shortlist of common events in DOM manipulation, but there are many more.

Event What it does
onclick An HTML element has been clicked on
onmouseover An HTML element was hovered over
onmouseout Mouse cursor moves off HTML element
onkeydown A keyboard button is pressed
onload The HTML page has finished loading
onchange Some HTML element has been modified
