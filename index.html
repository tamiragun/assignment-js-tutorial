JavaScript refresher

So you've learned JavaScript before, but have been using other languages and frameworks in the meantime? Perhaps all you need is a quick refresher, but most JavaScript courses are geared towards complete beginners. That's where this tutorial comes in.

This website covers the basics of JavaScript for people who already know programing fundamentals. And instead of focusing on the theory, it lets you jump right in and practise your skills.

Practising your skills

Next to each code block is a mini code editor where you can put the example into practise. To see your code ina ction, open your DevTools console. Right click on your browser and select "inspect", then click on the tab "console". The output of your statements will appear there.

To clear the code editor, you can press the "reset" button. To clear the console, right click in the console and click "Clear console".
